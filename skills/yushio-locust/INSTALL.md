# 蝗虫夕潮 · 安装与上手（给朋友）

蝗虫夕潮是一套让 AI **全包你的远程求职**的工具：从简历、给 HR 的建联消息，到面试押题和上场前 5 分钟速记——目标是让你**照着它准备的文档念，就能应付大多数面试**。

支持 **Claude Code** 和 **Codex** 两种环境，二选一即可。

---

## 一、安装

### 如果你用 Claude Code

把整个 `yushio-locust` 文件夹放进你的 skills 目录：

- **macOS / Linux**：`~/.claude/skills/yushio-locust/`
- **Windows**：`%USERPROFILE%\.claude\skills\yushio-locust\`

放好后新开一个 Claude Code 会话即可（它会自动发现这个 skill）。验证：随便开个会话说"你是蝗虫夕潮"，它应该进入求职副驾模式。

### 如果你用 Codex

Codex 读不到 Claude 的 skill 目录，改用项目内入口文件：

1. 在电脑上建一个**求职文件夹**（例如 `~/jobhunt`）。
2. 把 `yushio-locust/AGENTS.template.md` 复制进去，**改名为 `AGENTS.md`**。
3. 用 Codex 打开这个文件夹，它读到 `AGENTS.md` 就进入蝗虫夕潮模式。

---

## 二、第一次上手（三步）

1. **打开你的工具**：Codex 用户进你刚放了 `AGENTS.md` 的求职文件夹；Claude Code 用户随便在哪打开都行（它会主动帮你建求职文件夹，不用你先建）。
2. **唤醒**：说一句"**你是蝗虫夕潮**"（或直接"帮我开始找工作"）。
3. **跟着它走**：它会问你两件事——
   - 把你**过去的简历 / 履历**发给它（复制粘贴、PDF、md 文件、在线 profile 链接都行，越多越好）；
   - 告诉它你**本地哪个文件夹放着你做过的项目**（代码仓库 / 设计稿 / 文档），它会去读你**真正做过什么**，比你自己写得更准。

之后它会帮你建好档案、生成简历主稿和求职渠道清单。**每当你找到一个岗位**，把 JD（职位描述）发给它，它就产出那个岗位的全套材料。

---

## 三、每个岗位你会拿到什么

放在 `applications/日期_公司_岗位/` 下，共 5 个文件：

| 文件 | 作用 |
|---|---|
| `jd.md` | JD 存档 + 内部投递判断（自己看，不外发） |
| `resume_zh.md` / `resume_en.md` | 针对这个岗位的中 / 英文简历（可直接发出） |
| `hr_message.md` | 发给 HR 的第一条消息（中英 + 极简版） |
| `interview_prep.md` | **面试押题作答** + 顶部「⚡ 5 分钟速记区」，上场前扫一遍 |

---

## 四、隐私（重要）

- 你的简历、履历、薪资这些是**最敏感的个人数据**。它们**只存在你本地这个文件夹**，不会上传、不会分享。
- **不要**把求职文件夹 push 到公开的 GitHub 仓库。
- 这套工具本身（方法）可以随便分享给朋友，但**别把你自己的档案 / 简历当模板传给别人**。

## 五、找工作时的安全提醒

求职的人是诈骗高危目标。遇到陌生"招聘方"让你**下载安装某个 app、运行一段命令 / 脚本、连接钱包输助记词、交保证金"验证"**——几乎都是诈骗。正规招聘不会让你装来路不明的东西或动钱包。蝗虫夕潮看到这类信号也会提醒你。
