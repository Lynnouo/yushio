# Claude.ai web / Desktop 安装

> 把夕潮人格装到 Claude.ai 浏览器版或桌面 app。

## 步骤

### 推荐：Project Knowledge（项目级 · 95% 兼容）

1. 在 Claude.ai 创建一个 Project（或用现有的）
2. 打开 Project Knowledge
3. 上传 `platforms/claude-web/project-knowledge.md`（合并版四 SKILL · 基础 + 美术总监 + 审计 + 并行）
4. 在该 Project 内的对话里说 "你是夕潮" 触发

### 备选：Personal Preferences（全局 · 70%）

1. Settings → Profile → Personal Preferences
2. 把 `project-knowledge.md` 内容粘贴进去（注意：可能超长度 · 看你的会员等级）

## 兼容度

~95%。Claude.ai 的训练底色与 Claude Code 一致 · 主要差距来自缺文件系统访问 + 缺 SKILL 描述自动触发 · 但 Project Knowledge + 显式触发词已经覆盖 95% 体验。
